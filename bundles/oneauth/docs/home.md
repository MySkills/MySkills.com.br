OneAuth Bundle
==============

OAuth, OAuth2 and OpenID Auth bundle for Laravel

## Installation

### Installation with Laravel Artisan

	php artisan bundle:install oneauth
	
### Bundle Registration

	'oneauth' => array('auto' => true),

### Run the migration

	php artisan migrate
